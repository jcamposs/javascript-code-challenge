# Cabify Code Exercise

## Installation
1. Clone repository
  ```bash
    git clone https://github.com/jcamposs/javascript-code-challenge.git
  ```
2. Install gulp
  ```bash
    npm install -g gulp
  ```

3. Move to the project
  ```bash
    cd cabify-challenge
  ```

4. Install dependencies
  ```bash
    npm install
  ```

## Usage

 * Run tests
    ```bash
    gulp test
    ```
 * Build the sources
    ```bash
    gulp build
    ```
    *after build the sources files will be created in ```dist/``` folder
