# Cabify Code Exercise

## Installation
1. Install gulp
  ```bash
  npm install -g gulp
  ```

2. Move to the project
  ```bash
  cd cabify-challenge
    ```

3. Install dependencies
  ```bash
  npm install
  ```

## Usage

 * Run tests
    ```bash
    gulp test
    ```
 * Build the sources
    ```bash
    gulp build
    ```
    *after build the sources files will be created in ```dist/``` folder
